How to

วิธีซ้อมแต่ละแบบ

STRIDES

สไตรด์

คืออะไรวิ่งเร่งแบบผ่อนคลายสั้นๆ ~20 วินาที (80–100 ม.) ไม่ใช่สปรินต์สุดแรง
ทำยังไงเริ่มจากจ๊อก แล้วค่อยๆ เร่งขึ้นจนถึง ~90% ช่วง 5 วิสุดท้าย แล้วผ่อนลงเอง ฟอร์มผ่อนคลาย ไหล่ไม่เกร็ง
พักเดิน/ยืน 45–60 วินาที จนหายเหนื่อยสนิทก่อนเที่ยวถัดไป
เมื่อไหร่ท้าย easy run
ทำไมฝึกฟอร์ม เพิ่มความเร็วขาและความประหยัดแรง โดยไม่ทำให้ล้า — เหมาะกับคุณช่วงสร้างฐาน
INTERVALS

อินเทอร์วัล

เพซ7:00–7:35 /กม. หนักแต่คุมได้ ไม่สุดแรง
รอบละ800 ม. ≈ 5:40–5:50/เที่ยว · 1 กม. ≈ 7:10–7:15 · 400 ม. ≈ 2:45 · หรือจับเป็นนาที เช่น 3 นาที
พักจ๊อกเบาๆ (ไม่หยุดยืน) 90 วิ–2 นาที ให้หายใจกลับมาเกือบปกติ
วอร์มจ๊อกเบา 10 นาทีก่อน + คูลดาวน์ 10 นาทีหลัง เสมอ
เช็กถ้าเที่ยวท้ายๆ เพซตกเยอะ = ออกแรงแรกๆ เร็วเกินไป คุมให้ทุกเที่ยวเพซใกล้กัน
TEMPO

เทมโป / เธรชโฮลด์

เพซ7:30–7:50 /กม.
ความรู้สึก"สบายแต่หนัก" พูดได้แค่คำสองคำ · พูดเป็นประโยคได้ = ช้าไป · พูดไม่ได้เลย = เร็วไป
รูปแบบต่อเนื่อง 15–25 นาที หรือแบ่งชุด เช่น 2×12 นาที พัก 2 นาที
ทำไมยกระดับ lactate threshold — ตัวชี้วัดสำคัญที่สุดของฮาล์ฟ
RACE PACE

เพซแข่ง (HMP)

เพซ8:00 /กม. (สำหรับเป้า 2:50)
เมื่อไหร่Phase 3 เป็นต้นไป — ทั้งในเซสชันแยก และท้าย long run
ทำไมจุดอ่อนของคุณคือคุมเพซไม่นิ่ง การซ้อมเพซนี้ซ้ำๆ ให้ขา "จำ" ได้ คือกุญแจทำเวลา
LONG RUN

ลองรัน

เพซเริ่ม 8:30–9:00 · Phase 2–3 ท้ายๆ เร่งเข้าเพซแข่ง 8:00
กฎเหล็กออกช้า — เริ่มเร็วจะพังท้ายเส้น (เหมือน split เดิมของคุณ)
เติมพลังPhase 3 เริ่มซ้อมกินเจลทุก ~45 นาที + จิบน้ำ
ทำไมเซสชันสำคัญที่สุดของฮาล์ฟ สร้างความอึดของกล้ามเนื้อและระบบพลังงาน
EASY / STEADY

วันสบาย

Easy8:45–9:15 — ฟื้นตัว ยิ่งช้ายิ่งฟื้น วิ่งช้าได้ ไม่ผิด
Steady8:15–8:45 — สร้างฐาน aerobic สม่ำเสมอ
ห้ามแอบเร่งวันพวกนี้ ต้องรู้สึกว่า "ยังวิ่งต่อได้อีกเยอะ" เก็บแรงไว้วันคุณภาพ
Strength

เวทด้วยดัมเบล 2 อัน

2 ครั้ง/สัปดาห์ · 20–30 นาที · ทำ 2–3 รอบ · หลัง easy run หรือวันว่าง (อย่าทำหนักก่อน long run/วันคุณภาพ)

1
Goblet squat×12
ถือดัมเบลแนบอก ย่อลงลึก หลังตรง — ต้นขา/สะโพก
2
Romanian deadlift×12
ถือดัมเบล 2 ข้าง พับสะโพก เข่างอนิด — แฮมสตริง/หลังล่าง พลังขับสำคัญของนักวิ่ง
3
Reverse lunge×10/ข้าง
ก้าวถอยหลังย่อ ถือดัมเบล — ขาเดี่ยว/ทรงตัว
4
Calf raise×15
เขย่งปลายเท้าช้าๆ — เสริมน่อง/ข้อเท้าให้ทน
5
Tibialis raise×15
สำคัญสำหรับคุณ: พิงกำแพง ยกปลายเท้าขึ้นค้าง — กล้ามหน้าแข้ง ป้องกันอาการปวดที่เคยเป็น
6
Step-up / ยืนขาเดียว30 วิ/ข้าง
ความมั่นคงข้อเท้า/สะโพก
7
Core
Plank 40 วิ · Side plank 30 วิ/ข้าง · Dead bug 10/ข้าง — แกนกลางช่วยให้ฟอร์มไม่พังตอนล้า
Fuel

โภชนาการ + อาหารเสริม

ครีเอทีน
3–5 ก./วัน ทุกวันสม่ำเสมอ เวลาไหนก็ได้ ดื่มน้ำเยอะๆ · ช่วยงานเวท+ฟื้นตัว ปลอดภัย
เวย์โปรตีน
เสริมโปรตีนให้ครบวัน (~1.6–2.0 ก./นน.ตัว 1 กก.) กินหลังวิ่งหรือเป็นมื้อว่าง
ก่อน long run
กินคาร์บให้พอ อย่าวิ่งยาวท้องว่าง
ซ้อมกินเจล
เริ่ม Phase 3 · ทุก ~45 นาที ในวันยาว ให้ท้องชินก่อนแข่ง
อากาศ กทม. ร้อน — จิบน้ำ + เกลือแร่ระหว่างวิ่งยาวเสมอ อย่ารอจนกระหาย
Injury

ป้องกันบาดเจ็บ

  • เลี่ยงรองเท้าคาร์บอนในซ้อมและแข่ง — ต้นเหตุที่คุณปวดข้อเท้า/หน้าแข้ง แผ่นคาร์บอนเปลี่ยนการลงเท้า ใช้เทรนเนอร์ที่คุ้นเคยปลอดภัยกว่ามาก
  • ห้ามลองอะไรใหม่ในวันแข่ง — รองเท้า เสื้อผ้า เจล อาหาร ต้องผ่านการซ้อมมาแล้ว
  • เพิ่ม cadence 160 → 170 ก้าว/นาที ก้าวสั้นถี่ขึ้น ลดแรงกระแทกหน้าแข้ง/เข่า
  • ทำ Tibialis raise สม่ำเสมอ เป็นเกราะกันปวดหน้าแข้งโดยตรง
  • ปวดหน้าแข้ง/ข้อเท้าเมื่อไหร่ = ลดโหลดทันที ประคบเย็น อย่าฝืน ไม่ดีขึ้นใน 3–4 วันควรพบผู้เชี่ยวชาญ

Race day

กลยุทธ์วันแข่ง (ตี 2)

งานปล่อยตัว 02:00 น. ต้องเตรียมต่างจากงานเช้าปกติ

  • 2 วันก่อน: เพิ่มคาร์บ (ข้าว เส้น ขนมปัง) ลดปริมาณวิ่ง พักขา
  • เสาร์กลางวัน: งีบช่วงบ่าย–เย็นให้ได้ เพราะต้องตื่นตลอดคืน
  • ~23:00–24:00 (2–3 ชม.ก่อนสตาร์ท): มื้อเบาย่อยง่าย คาร์บสูง (กล้วย ขนมปัง ข้าวต้ม) เลี่ยงของมัน/ไฟเบอร์สูง/ของใหม่
  • คาเฟอีน: ถ้าเคยชิน กินก่อนสตาร์ท ~45–60 นาที
  • ซ้อมวิ่งกลางคืน ตั้งแต่ Phase 3 อย่างน้อย 2–3 ครั้ง
  • เพซ (สำคัญสุด): 3–4 กม.แรก วิ่งช้ากว่าเป้า 10–15 วิ/กม. แล้วเร่งเข้าเป้าครึ่งหลัง (negative split)
  • ระหว่างแข่ง: กินเจลราวๆ กม.7 และ กม.14 · จิบน้ำทุกจุด
